# 🚚 Delivery System

Sistema de delivery desenvolvido em Java com foco no gerenciamento de pedidos, clientes e entregas.

## 📋 Sobre o Projeto

O Delivery System é uma aplicação desenvolvida para simular o funcionamento de uma plataforma de delivery, permitindo o controle de pedidos, entregadores e clientes de forma simples e organizada.

Projeto desenvolvido para fins acadêmicos/práticos utilizando Java.

---

## ⚙️ Funcionalidades

- ✅ Cadastro de clientes
- ✅ Cadastro de pedidos
- ✅ Gerenciamento de entregas
- ✅ Controle de status do pedido
- ✅ Organização das informações do sistema
- ✅ Estrutura orientada a objetos

---

## 🛠️ Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos (POO)
- IntelliJ IDEA
